# Data Directory

This folder should contain the following data files after running the scraping scripts:

## Expected Files

| File | Description | Generated By |
|------|-------------|--------------|
| `treatwell-all-{date}.csv` | Full dataset with raw API responses | `scraping/crawl-treatwell.py` |
| `treatwell_without_raw-all-{date}.csv` | Cleaned dataset (without raw JSON) | `scraping/crawl-treatwell.py` |
| `treatwell_kids-{date}.csv` | Children's haircuts with raw responses | `scraping/crawl-treatwell.py` |
| `treatwell_without_raw_kids-{date}.csv` | Children's haircuts (cleaned) | `scraping/crawl-treatwell.py` |
| `venue_info-{date}.csv` | Venue metadata (employee count, type) | `scraping/scrape-venue-info.py` |
| `treatment_ids_analysis.csv` | Treatment category analysis | `analysis/analyze_treatment_ids.py` |

## Data Availability

Due to Treatwell's Terms of Service, we cannot publicly share the scraped data. However, researchers can replicate our data collection using the provided scraping scripts.

## Replication Instructions

1. Run `scraping/crawl-treatwell.py` to collect haircut pricing data
2. Run `scraping/scrape-venue-info.py` to collect venue metadata
3. Run `analysis/analyze_treatment_ids.py` to analyze treatment categories
4. Use the Jupyter notebooks in `analysis/` for statistical analysis
